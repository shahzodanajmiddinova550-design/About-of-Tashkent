# About-of-Tashkent
Toshkent – O‘zbekistonning eng yirik shahri va poytaxti, mamlakatning siyosiy, iqtisodiy, madaniy va ilmiy markazi hisoblanadi. Bu shahar Markaziy Osiyoning yuragida joylashgan bo‘lib, o‘tmishda boy tarixga ega va zamonaviy rivojlanishning namunasi sifatida kelajakda yanada ko‘proq o‘zgarishlarni kutmoqda.
